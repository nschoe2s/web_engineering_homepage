10. Vue.js

Vue (pronounced /vjuː/, like view) is a progressive framework for building user interfaces.
Inhalt:
Vorlesung 10
Lernziele:
  1. Vue-Instanzen konfigurieren können.
  2. Vue-Komponenten schreiben können.
  3. Single File Vue Components erstellen können.
  4. SVG-Grafiken mit Interaktion versehen können.
  5. Mit Vue SPAs erstellen können.
Videos, Tutorials und Blogs:
  • Vue.js Guide
  • Vue.js API
  • Vue CLI
  • Vue Cookbook
  • Vue Examples
  • Traversy Media: Vue JS Crash Course - 2019 In this crash course you will learn all about Vue.js including what it is, Vue-CLI, components, data, events, directives, etc.
Mögliche Klausuraufgaben:
  • Vergleichen Sie Vue.js mit LitElement. Was sind die wesentlichen Unterschiede? Was sind die jeweiligen Vor- und Nachteile?
  • Ergänzen Sie ein Vue.js-Programm (als Lückentext) um die passenden Vue.js-Direktiven.
  • Schreiben Sie eine Vue.js-Komponente Counter, die einen Button zum Inkrementieren besitzt und den Zählerstand anzeigt.
  • Schreiben Sie eine Vue.js-Komponente Switch, die einen Button zum An- und Ausschalten besitzt und damit eine Lampe ein- und ausschaltet.


10.0 Vue-Tutorial
  • Arbeiten Sie die Essentials im Vue-Guide durch. Ein aktuelles Vue-Tutorial finden Sie auf Youtube. Den Code für das Tutorial finden Sie auf GitHub.
  • Hinweise zum Debugging lesen Sie im Vue.js Cookbook unter Debugging in VS Code. Ein Erklärvideo dazu finden Sie auf Youtube unter 3 Ways To Debug Your Vuejs Apps With VS Code And Chrome.

10.1 Counter in Vue.js (2 Punkt)
Schreiben Sie eine Vue.js-Komponente mit 2 Buttons zum Inkrementieren und Dekrementieren eines Counters. Angezeigt werden soll nicht nur der aktuelle Stand des Counters, sondern auch, wie oft geklickt wurde. Betten Sie Ihre Counter-Komponente in die Webseite zweimal ein und testen Sie, ob beide Counter unabhängig voneinander sind.

10.2 Menü-Komponente (3 Punkte)
Schreiben Sie eine möglichst flexible Vue.js Component für Menüs und wenden Sie diese in derselben Webseite zweimal an.

10.3 Vue.js WWW-Navigator (5 Punkte)
Schreiben Sie Ihren WWW-Navigator als SPA in Vue.js.
Dokumentieren Sie Ihren Entscheidungsprozess: In welche Komponenten wollen Sie Ihre App zerlegen? Wie soll das State Management implementiert werden?
