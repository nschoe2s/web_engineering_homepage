11. MEVN

Full-Stack Web Development mit dem MEVN-Stack: MongoDB, Express.js, Vue.js und Node.js
Inhalt:
Vorlesung 11
Lernziele:
  1. Full-Stack Web Development beschreiben können.
  2. Full-Stack Web-Architekturen nennen und beschreiben können.
  3. Eine Full-Stack Web-Anwendung eigenständig mit MEVN umsetzen können.
Videos, Tutorials und Blogs:
  • Youtube-Videos:
    • Full Stack Vue.js, Express & MongoDB [1] - Express API Traversy Media
    • MEVN Stack - Project WalkThrough @Youtube
  • Tutorials:
    • Build full stack web apps with MEVN Stack
    • Creating a ToDo App with a MEVN Full Stack
  • MEVN-CLI, Light Speed setup for MEVN stack based apps
  • Full Stack Web Development with Vue.js and Node.js, Packt Publishing 2019.
Mögliche Klausuraufgaben:
  • Beschreiben Sie den MEVN-Stack.
  • Beschreiben Sie die Event-Queue von Node.js.
  • Erläutern Sie den Unterschied zwischen Blocking I/O und Non-Blocking I/O (a) prinzipiell (b) konkret anhand von einfachen Code-Beispielen.
  • Beschreiben Sie die Besonderheiten von MongoDB.
  • Ergänzen Sie einen Lückentext um MEVN-Code.


11.0 MEVN-Tutorial
  • Arbeiten Sie das MEVN-Tutorial 1 und 2 durch.
  • Den Code für das Tutorial finden Sie auf GitHub. Hinweis: Löschen Sie package-lock.json, da die darin festgehaltenen npm-Pakete teilweise nicht mehr existieren, so dass es zu einem Fehlerabbruch kommt.

11.1 Erweiterung der MEVN-App (1 Punkt)
Erweitern Sie die MEVN-App aus dem Tutorial um einen Delete-Button je Todo-Item. Außerdem soll die Eingabe eines neuen Posts auch mit der Enter-Taste erfolgen können.

Für diese Übungsaufgabe habe ich keine zufriedenstellende Lösung herausgearbeitet.

11.2 MEVN WWW-Navigator (9 Punkte)
Schreiben Sie Ihren WWW-Navigator auf dem MEVN-Stack. Erweitern Sie den Navigator um eine Eingabemöglichkeit, mit der neue Einträge und Menüpunkte hinzugefügt werden können. Speichern Sie diese persistent in der MongoDB-Datenbank. Eine Lösung zu der Aufgabe 10.3 finden Sie auf GitHub.
Für diese Übungsaufgabe habe ich keine zufriedenstellende Lösung herausgearbeitet.
