2.1 Fachliche Argumentation über Erfolgsprinzipien des WWW (2 Punkte)
Mit welchen fachlichen Argumenten wurde das WWW-Proposal von TBL abgelehnt?
Tim Berners-Lees Chef schrieb zu seiner Idee: "Wage, aber spannend." und die Hypertext- Konferenz verstand nicht, was daran nützlich sein sollte.
Was sind die fachlichen Argumente, warum das WWW dennoch ein Erfolg wurde?
Als Tim Berners-Lee sein Konzept-Papier bei der Hypertext-Konferenz einreicht, wird er abgelehnt, bekommt jedoch eine Erlaubnis zu einer Poster-Session. Dort wird er dann von Marc Andreesen entdeckt. Nur deshalb wurde das WWW dennoch ein Erfolg.
Was wäre der Preis für die garantierte Verhinderung von "broken links"?
Kein WWW.

2.2 HTML-Literatur lesen und Fragen beantworten (2 Punkte)
Arbeiten Sie ein HTML-Tutorial durch ( W3 Schools oder selfhtml oder André Kless ) und beantworten Sie dazu folgende Fragen:
Was ist HTML?
HTML (Hypertext Markup Language) ist die textbasierte Standardauszeichnungssprache für Webseiten zur Strukturierung elektronischer Dokumente wie Texte mit Hyperlinks, Bildern und anderen Inhalten. HTML dient dazu, einen Text semantisch zu strukturieren, nicht aber zu formatieren. Die visuelle Darstellung ist nicht Teil der HTML-Spezifikationen und wird durch den Webbrowser und Gestaltungsvorlagen wie CSS bestimmt. Mit HTML kann man seine eigene Website erstellen.
Wie kann man eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen?
Man kann eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen, indem man die HTML-Elemente <ol> und <li> in drei Ebenen Schachtelt wie z.B. in folgendem Beispiel:
<!DOCTYPE html>
<html lang="de">
 <head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Übung 2 Aufgabe 2</title>
 </head>
 <body>
  <h3><u>Geschachtelte geordnete Liste der Schachtelungstiefe 3</u></h3>
  <ol>
   <li> Erster Punkt Ebene 1 </li>
   <ol>
    <li> Erster Punkt Ebene 2 </li>
    <ol>
     <li> Erster Punkt Ebene 3 </li>
    </ol>
   </ol>
  </ol>
 </body>
</html>
Wie ist eine HTML-Tabelle aufgebaut?
Eine HTML-Tabelle besteht aus den HTML-Elemente <table></table> als Rahmen für die gesamte Tabelle, <tr></tr> als Rahmen für eine Tabellenzeile, <th></th> als Inhalt einer Zelle allerdings im Format einer Tabellenüberschrift (Zelleninhalt in bold) und <td></td> als Inhalt einer Zelle.
Beispiel einer Tabelle bestehend aus zwei Zeilen und drei Spalten:
<table>
 <tr>
  <th>Zeile 1 Spalte 1 in bold</th>
  <th>Zeile 1 Spalte 2 in bold</th>
  <th>Zeile 1 Spalte 3 in bold</th>
 </tr>
 <tr>
  <td>Zeile 2 Spalte 1</td>
  <td>Zeile 2 Spalte 2</td>
  <td>Zeile 2 Spalte 3</td>
 </tr>
</table>
Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?
Man sollte für eine gute Übersicht Verzeichnisse in Unterverzeichnisse einordnen. Dem Hauptverzeichnis sollte man den Namen index.html geben und man sollte die Namen durchgehend mit Kleinbuchstaben erstellen und nicht mit Großbuchstaben mischen.
Wie baut man in HTML ein Menü?
Ein HTML-Menü wird als erstes im Body geschrieben und wird innerhalb eines nav-Elements platziert, damit man es sofort sieht. Es besteht aus einer untergeordneten Liste. Die einzelnen List-Items werden mit Verlinkungen versehen, wobei die Seite, auf der man sich grade befindet, mit einem ARIA-Attribut versehen. Ein ARIA-Attribut ist ein Leereintrag, der Screenreadern anzeigt, auf welcher Seite sich der Benutzer befindet.
Beispiel:
Seite 1:
<!DOCTYPE html>
<html lang="de">
 <head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Übung 2 Aufgabe 2</title>
 </head>
 <body>
  <nav>
   <ul>
    <li><a tableindex=“0“ aria-current=“page“>Seite 1</a></li>
    <li><a href=“seite2.html“>Seite 2</a></li>
   </ul>
  </nav>
 </body>
</html>
Seite 2:
<!DOCTYPE html>
<html lang="de">
<head>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Übung 2 Aufgabe 2</title>
</head>
<body>
<nav>
<ul>
<li><a href=“seite1.html“>Seite 1</a></li>
<li><<a tableindex=“0“ aria-current=“page“>Seite 2</a></li>
</ul>
</nav>
</body>
</html>
Welche Attribute sollte man bei Bildern wie verwenden?
Um ein Bild einzubinden braucht man das HTML-Element <p></p> und die Attribute img, src, alt und optional auch width und heigth.
Beispiel ohne width und heigth:
<p><img src=“bett.jpg“ alt=“Bett“></p>
Beispiel mit width und heigth:
<p><img src=“bett.jpg“width=“201“ heigth=“216“ alt=“Bett“></p>
