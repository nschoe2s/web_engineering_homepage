9. WebApps

Das WWW als App-Plattform
Inhalt:
Vorlesung 9
Lernziele:
  1. Beschreiben können, was dem Web als App-Plattform fehlt.
  2. Web Application Architectures beschreiben können.
  3. Multi-Threading im WWW implementieren können.
  4. Material Design Web Component nutzen können.
  5. Router für SPAs und MPAs schreiben können.
  6. RESTful Routes entwickeln können.
  7. Location API und History API für Routing nutzen können.
  8. Das SAM-Pattern anwenden können.
  9. Funktionskomponenten mit Hooks entwickeln können.
  10. State Management mit Redux entwickeln können.
